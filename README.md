# Бот записи на посещение (MAX Messenger)

Бот для записи на день открытых дверей в школе. Работает в MAX Messenger.

## Возможности

- Запись на свободные даты и время
- Админ-панель: добавление/удаление слотов, удаление записей, экспорт
- Напоминания за 24 и 2 часа до посещения
- Автоочистка прошедших слотов

## Требования

- Python 3.9+
- MAX Messenger API (токен бота)

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `config.yaml` и отредактируйте:

   - **bot.token** — токен бота MAX (получить в [dev.max.ru](https://dev.max.ru))
   - **admin_ids** — user_id администраторов (через `/myid` в боте)

2. Файл БД создаётся автоматически (`bot.db` по умолчанию).

## Запуск

```bash
python bot.py
```

С указанием своего конфига:

```bash
python bot.py /path/to/config.yaml
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и начало записи |
| `/myid` | Показать ваш user_id (для добавления в admin_ids) |
| `/admin` | Админ-панель |

## Структура конфига

- **bot.token** — токен MAX
- **database.path** — путь к SQLite-файлу
- **admin_ids** — список user_id админов
- **messages** — тексты сообщений
- **slots** — формат даты/времени
- **cleanup** — автоочистка прошедших слотов
- **reminders** — напоминания
- **export** — формат экспорта (xlsx/csv)
